# Текстовая приключенческая игра "Лабиринт сокровищ" | Иван Кириян | М25-555

"Лабиринт сокровищ" - текстовая консольная игра. В ней пользователи могут исследовать 8 комнат:

* Стартовая комната
* Большой зал
* Комната с ловушкой
* Библиотека
* Оружейная комната
* Комната с сокровищем
* Комната воспоминаний
* Комната мечты

В комнатах пользователи могут использовать различные предметы, подбирать их, обходить ловушки и решать загадки. Игра усложнена с помощью генерации случайных событий и ловушек.

# Как поиграть?

1) Скопируйте проект локально. Через кнопку code в шапке репозитория.
2) Выполните команду
```make install```
3) Затем
```make project```

# Вам доступны следующие команды:

```text
go <direction>   - перейти в направлении (north/south/east/west)

north/south/east/west - перейти в направлении без go

look             - осмотреть текущую комнату

take <item>      - поднять предмет

use <item>       - использовать предмет из инвентаря

inventory        - показать инвентарь

solve            - попытаться решить загадку в комнате

quit             - выйти из игры

help             - вывести вспомогательные команды
```
